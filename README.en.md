# system_monitor

#### Description
监控windows系统的事件，包含内核事件（进程，线程，文件，网络，句柄，内核对象，内存，ALPC等）、用户态事件（RPC等）。相比procmon，可以监控更多的事件，过滤功能更强大，更实用。
